# Projet_10
Site vitrine de 724events, agence évènementielle.

## 724 Events

Ce projet vise à débugger une application web pour la gestion d'événements en utilisant React.

## Aperçu

![Capture d'écran de la page web](https://i.ibb.co/kBx0VM0/724events.png)

## Pré-requis

Avant de commencer à travailler sur ce projet, assurez-vous d'avoir les éléments suivants installés sur votre système :

- Node.js (version >= 16.14.1 LTS)
- npm (version >= 8.5.5)
- Yarn

## Installation
- `yarn install`

## Lancement de l'application
- `yarn start`

## Tests
- `yarn test --watch`
